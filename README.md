# QA Playwright Portfolio

This repository is a QA automation portfolio built with Playwright and TypeScript. It demonstrates functional UI testing, cross-browser execution, report generation, and reusable automation practices.

## What is included

- Playwright test suite in `tests/`
- Cross-browser configuration for Chromium, Firefox, and WebKit in `playwright.config.ts`
- HTML reports generated by Playwright in `playwright-report/`
- Example test cases covering basic UI flows and validation

## Why this portfolio

This repo is designed for QA/automation roles and shows that I can:

- write stable Playwright tests using modern TypeScript
- structure tests and configuration for maintainability
- run tests in multiple browsers
- generate and share test reports
- document how to reproduce automation locally

## Tech stack

- Playwright
- TypeScript
- Node.js / npm
- HTML test reporting

## Setup

1. Install dependencies

```bash
npm install
```

2. Install Playwright browsers

```bash
npx playwright install
```

## Run tests

Run the full suite:

```bash
npm test
```

Run a single test file or folder:

```bash
npx playwright test tests/example.spec.ts
```

## View reports

After the tests complete, open the HTML report:

```bash
npx playwright show-report
```

There is also a generated report folder at `playwright-report/`.

## Test structure

- `tests/` — contains Playwright test files
- `playwright.config.ts` — Playwright configuration and browsers
- `playwright-report/` — saved report assets from test runs
